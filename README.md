# AI-Based Facial Recognition Attendance System

An advanced, real-time facial recognition attendance system built using:
- **Frontend**: React, Vite, TailwindCSS, Shadcn/ui
- **Backend & Database**: Supabase (Auth, Storage, Database Triggers)
- **AI Vision Engine**: Google Gemini API (`gemini-3.1-flash-lite`)

## Deployment Setup
1. Configure `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` environment variables.
2. Deployed Deno Edge Functions on Supabase.
